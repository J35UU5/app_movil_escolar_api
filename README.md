#  API de Gestión de Eventos Académicos (Backend)

Backend desarrollado con **Django REST Framework** para el sistema de gestión escolar. Esta API maneja la autenticación de usuarios, la lógica de negocio para el CRUD de eventos académicos y la generación de estadísticas para gráficas.

##  Información del Proyecto
- **Materia:** Desarrollo de Aplicaciones Móviles (Otoño 2025)
- **Institución:** Benemérita Universidad Autónoma de Puebla (BUAP)
- **Facultad:** Ciencias de la Computación
- **Profesor:** M.C. Luis Yael Méndez Sánchez

##  Tecnologías Utilizadas
- **Lenguaje:** Python 3.x
- **Framework:** Django & Django REST Framework
- **Base de Datos:** SQLite (Desarrollo) / PostgreSQL (Producción en Render)
- **Librerías Clave:**
  - `django-cors-headers` (Manejo de CORS)
  - `psycopg2-binary` (Conexión a PostgreSQL)
  - `gunicorn` (Servidor de producción)
